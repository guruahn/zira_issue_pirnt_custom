# ZIRA 이슈 3열로 프린트
 - 지라의 이슈를 프린트할 경우 2열로 출력되는 것을 3열로 수정하고 가독성을 향상시킴.
 - 이슈에 외곽선을 넣고, 옵션으로 색상을 변경할 수 있도록 함.
